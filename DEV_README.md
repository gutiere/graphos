# Graphos

## Local Installation

-   [Install UV](https://docs.astral.sh/uv/getting-started/installation/)
-   Install dependencies
    -   `$ uv sync`

## Running Locally

-   `$ uv run -m graphos.src.main`

## Deploy to https://pypi.org/

-   `$ sh scripts/deploy.sh`
